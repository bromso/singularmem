# `@bromso/singularmem-linux-arm64-gnu`

This is the **aarch64-unknown-linux-gnu** binary for [singularmem](https://www.npmjs.com/package/singularmem).

This package is automatically installed by `npm install singularmem` on ARM64 Linux systems with glibc. You should not install this package directly; use the main `singularmem` package instead.
