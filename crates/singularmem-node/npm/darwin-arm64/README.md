# `@bromso/singularmem-darwin-arm64`

This is the **aarch64-apple-darwin** binary for [singularmem](https://www.npmjs.com/package/singularmem).

This package is automatically installed by `npm install singularmem` on Apple Silicon Macs. You should not install this package directly; use the main `singularmem` package instead.
