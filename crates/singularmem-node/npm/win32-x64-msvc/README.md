# `singularmem-win32-x64-msvc`

This is the **x86_64-pc-windows-msvc** binary for [singularmem](https://www.npmjs.com/package/singularmem).

This package is automatically installed by `npm install singularmem` on Windows x86_64 systems. You should not install this package directly; use the main `singularmem` package instead.
